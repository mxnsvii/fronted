Form tag

Method of form tags

  1. GET
  2. POST


1) GET Method

GET will send form data to the server using URL by appending URL ?

? will be called as query string

Example file:///c%3A/xampp/htdocs/FrontEnd/26jun_frontend_MWF_8/HTML/27Form/01Intro.html?username=asdf&fullname=asdfdd

its shows its data in url that's why get method is less secure

GET is default method of form

GET is faster than post

GET mainly we use it for searching

send one page data to other with refreshing

we can bookmark URL with data

limitation 2048 char only

not supporting file uploading



2) POST Method

POST method sends its data without using url

secure

no limitation

not able to do bookmark

support file uploading
